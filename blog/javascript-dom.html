JavaScript and its relationship to HTML and CSS... for humans

When creating websites, we're using three major components: HTML (a markup language), CSS (a styling language), and an actual programming language, JavaScript. One way to describe the relationship between this iconic trio is to compare it to the human body.

HTML is responsible for the overall structure and content of a website. It provides our imaginary body with all of the bones, working organs and limbs. For now, the body is alive but not yet moving or speaking.

CSS makes the websites pretty and positions elements according to the designer's preference. This is similar to choosing suitable clothes, a certain colour of nail polish and adding a pair of earrings.

Finally, JavaScript makes the page interactive and brigs them to life. In terms of the body analogy, JavaScript provides us with functionality – at this point we can communicate and move.

Control flow and loops... for alien forms

Control flow is how a program is executed, and normally it reads top to bottom. Imagine any everyday action you don't think twice about — going to the store, picking out an outfit for the day, walking your dog. If you had to explain this process to a friendly alien, you would pause and then, hopefully, come up with a set of step-by-step detailed instructions. Here's some imaginary code for making yourself a nice cuppa: 

        
function makeTea() {
    openKettle()
    pourWaterInKettle()
    closeKettle()
    turnKettleOn()
    waitForBoiling()
    getCup()
    pourWaterInCup()
    getTeabag()
    swirlTeabagInCup()
    takeTeabagOutOfCup()
    ???
    PROFIT
}

Well, that's fine, but there are so many situations in our day to day lives when we repeat something unless something else happens. That's when loops come in. Examples? Your local supermarket has been recently renovated, and all of your favourite foods are now in all the wrong places! Take a look at this pretend while loop: 

      
 while (cripsInMyBasket = 0) {
    keepRunningAroundSupermarket()
    hangry = True
  }
