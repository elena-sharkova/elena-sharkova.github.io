HTML and CSS

FADE IN:

EXT. BUILDING SITE - DAY

HTML relaxes in a sun lounger outside of a newly built house. CSS approaches.

CSS

What the…

HTML

Hey dude!

CSS

I just got off the phone with our client and had to see for myself.

HTML

Oh right! The house turned out pretty well, eh?

CSS

(stammering)

Are you out of your mind? Look around!

HTML

That’s what I call a job well done.

CSS

You’re freaking insane!!! All of the windows are lined up on the same wall! And they are… blue?!

HTML

I’m not sure I see the problem.

CSS

This is unlivable!

HTML

Did they not want the windows?

CSS

(exasperated)

Of course they wanted the damn windows!

HTML

Good news, buddy! They got ‘em!

CSS

You’re going to move the windows.

HTML winks.

HTML

Well, that’s your job, mate.

CSS

How am I even supposed to do it?!

HTML

I’m sure you’ll figure it out.

CSS takes a deep breath.

CSS

Fine. What about these giant piles of rocks along the property line?

HTML

Looks mint!

CSS

What is it even?!

HTML

Did they not want a fence?

CSS

I think I know where this is going.

HTML

(talks over CSS)

Good news, buddy! They got it!

CSS

I guess I will have to do the landscaping myself.

HTML

Glad we’re on the same page. The same page. You get it?

HTML breaks the “fourth wall” and obnoxiously winks at the audience. CSS sighs.

CSS

Okay then. So for tomorrow we’ve planned a visit by JavaScript.

HTML

Going… thr… a… tun…nel..

CSS

You do realise I’m standing right next to you.

HTML

Can’t…hear…

CSS

(screams)

You’re not even on the phone!
